# NumbersGame
A simple app to improve your mental math skills through quick, daily challenges.

![Screenshot](./assets/screenshot.png)

## About
Since finishing high school, I noticed that my mental math skills were getting worse. 
This project is my way of improving them again while also practicing app development. 
NumbersGame provides short exercises focused on speed and accuracy.


This README has been optimized for accessibility based on GitHub's blogpost "[Tips for Making your GitHub Profile Page Accessible](https://github.blog/2023-10-26-5-tips-for-making-your-github-profile-page-accessible)".

## Features

- Quick mental math challenges
- Increasing difficulty
- Time-based scoring
- Simple and minimal UI

## Installation (macOS)

1. Download the latest release from the Releases page
2. Open the .dmg file
3. Drag "NumbersGame.app" into your Applications folder
4. Launch the app

If macOS blocks the app:
- Go to System Settings → Privacy & Security
- Click "Open Anyway"
